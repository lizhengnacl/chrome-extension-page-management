# Chrome 页面管理插件

一个简单高效的 Chrome 页面管理工具，让您轻松组织和管理常用网页。

## 功能特性

### 🚀 快速保存页面
- 点击插件图标，一键保存当前浏览的网页
- 自动获取页面的标题、URL和图标
- 保存时可选择分组和添加标签

### 📁 分组管理
- 创建自定义分组（如"工作资料"、"学习资源"等）
- 支持分组的编辑和删除
- 一键打开分组内的所有页面

### 🏷️ 标签系统
- 创建彩色标签，为页面添加分类标记
- 支持为页面添加多个标签
- 通过标签快速筛选页面

### ⭐ 常用页面
- 将常用页面标记为星标
- 在新标签页最顶部展示，方便快速访问

### 🔍 智能搜索
- 通过关键词搜索页面（支持标题、URL、标签匹配）
- 快速定位需要的页面

### 💾 数据备份
- 导出所有数据为 JSON 文件
- 从 JSON 文件导入数据
- 轻松迁移和备份您的页面数据

## 安装方法

### 从源代码安装（开发者模式）

1. 下载或克隆此项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹，完成安装

## 使用说明

### 保存当前页面

1. 浏览想要保存的网页
2. 点击浏览器工具栏中的插件图标
3. （可选）为页面选择分组和添加标签
4. 点击"添加"按钮，页面即被保存

### 管理页面

打开新标签页，您会看到管理界面：

- **添加页面**：点击顶部的"添加页面"按钮，手动输入URL和标题
- **编辑页面**：点击页面卡片上的✏️图标进行编辑
- **删除页面**：点击页面卡片上的🗑️图标删除（需要确认）
- **标记常用**：点击⭐图标将页面添加到常用区域
- **拖拽排序**：在同一分组内拖拽调整页面顺序

### 管理分组

- **创建分组**：点击"添加分组"按钮，输入分组名称
- **编辑分组**：点击分组标题旁的"编辑"按钮
- **删除分组**：点击"删除"按钮（分组下的页面会变为未分组）
- **打开全部分页**：点击"打开全部"按钮，一次性打开分组内所有页面

### 管理标签

- **创建标签**：点击"添加标签"按钮，输入名称并选择颜色
- **编辑标签**：点击标签旁的"编辑"按钮修改
- **删除标签**：点击"删除"按钮（标签会从所有页面中移除）
- **筛选页面**：点击左侧标签，只显示带有该标签的页面

### 搜索页面

在顶部搜索框输入关键词，会实时匹配：
- 页面标题
- 页面URL
- 标签名称

### 数据导入导出

- **导出数据**：点击"导出"按钮，将所有数据保存为JSON文件
- **导入数据**：点击"导入"按钮，选择之前导出的JSON文件恢复数据

## 数据存储

所有数据存储在 Chrome 浏览器的同步存储空间中（`chrome.storage.sync`），会自动在您的设备间同步（需要登录 Chrome 账号）。
